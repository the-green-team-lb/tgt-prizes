# The Green Team Prizes

Provably fair prize draws. Every winner is determined by cryptographic verification.

## How It Works

1. **Order on LittleBiggy** - All orders placed with us on [LittleBiggy](https://littlebiggy.net/viewSubject/p/4775239) receive a raffle ticket from the 1st until the last day of the month
2. **Competitions & Giveaways Too** - All competitions and giveaways are run with numbers and drawn on this provably fair site
3. **Watch the Live Draw** - Go to the Live page when the draw starts. Watch the spinning wheel and see winners revealed
4. **Winners Are Announced** - Winning numbers are revealed one by one. Message us on [LittleBiggy](https://littlebiggy.net/viewSubject/p/4775239) with your winning ticket or number to claim your prize

## Provably Fair

- All numbers are loaded into the system before the draw
- A **SHA-256 hash** is published before the draw starts - locking in the fairness proof
- Winners are determined by a **cryptographic seed** committed before the draw - no one can change or predict results
- After the draw, the secret seed is revealed so anyone can verify the results
- Every completed draw shows the exact seed, numbers, and hash for independent verification

## Code Transparency

- Full source code is public on this GitHub repository
- Every code change is permanently recorded and visible in commit history
- Code is automatically hashed and timestamped on the **Bitcoin blockchain** via [OpenTimestamps](https://opentimestamps.org)
- This site has no contact forms, emails, or external links - all communication happens via [LittleBiggy](https://littlebiggy.net/viewSubject/p/4775239) only

## Tech Stack

- **Next.js** - React framework
- **Convex** - Real-time backend database
- **Tailwind CSS** - Styling
- **SHA-256** - Cryptographic hash verification
- **OpenTimestamps** - Bitcoin blockchain timestamping
